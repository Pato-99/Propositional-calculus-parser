# Propositional-calculus-parser
Parses formula from input and prints the truth table.

Representations:  literals -> any uppercase letter
                 
                  negation -> '!'
                  
                  konjunction -> '*'
                  
                  disjunction -> '+'
                  
                  implication -> 'i'
                  
                  equivalence -> 'e'
                 
Important: Input formula must be in reverse polish notation
